dc_event_t class

Opaque object describing a single event.

uint32_t dc_event_t::dc_event_get_account_id(dc_event_t* event)

Get account-id this event belongs to.

Parameters
event Event object as returned from dc_accounts_get_next_event().
Returns account-id belonging to the event, 0 for account manager errors.

The account-id is of interest only when using the dc_accounts_t account manager. To get the context object belonging to the event, use dc_accounts_get_account().

int dc_event_t::dc_event_get_data1_int(dc_event_t* event)

Get a data associated with an event object.

Parameters
event Event object as returned from dc_get_next_event().
Returns "data1" as a signed integer, at least 32bit, the meaning depends on the event type associated with this event.

The meaning of the data depends on the event-id returned as DC_EVENT constants by dc_event_get_id(). See also dc_event_get_data2_int() and dc_event_get_data2_str().

int dc_event_t::dc_event_get_data2_int(dc_event_t* event)

Parameters
event Event object as returned from dc_get_next_event().

char* dc_event_t::dc_event_get_data2_str(dc_event_t* event)

Get a data associated with an event object.

Parameters
event Event object as returned from dc_get_next_event().
Returns "data2" as a string or NULL. the meaning depends on the event type associated with this event. Once you're done with the string, you have to unref it using dc_unref_str().

The meaning of the data depends on the event-id returned as DC_EVENT constants by dc_event_get_id(). See also dc_event_get_data1_int() and dc_event_get_data2_int().

int dc_event_t::dc_event_get_id(dc_event_t* event)

Get the event-id from an event object.

Parameters
event Event object as returned from dc_get_next_event().
Returns once of the DC_EVENT constants. 0 on errors.

The event-id is one of the DC_EVENT constants. There may be additional data belonging to an event, to get them, use dc_event_get_data1_int(), dc_event_get_data2_int() and dc_event_get_data2_str().

void dc_event_t::dc_event_unref(dc_event_t* event)

Free memory used by an event object.

Parameters
event Event object as returned from dc_get_next_event().

If you forget to do this for an event, this will result in memory leakage.
