dc_array_t class

An object containing a simple array.

Contents

  • Reference
    • Public functions

This object is used in several places where functions need to return an array. The items of the array are typically IDs. To free an array object, use dc_array_unref().

int dc_array_t::dc_array_is_independent(const dc_array_t* array, size_t index)

Return the independent-state of the location at the given index.

Parameters
array The array object.
index Index of the item. Must be between 0 and dc_array_get_cnt()-1.
Returns 0=Location belongs to the track of the user, 1=Location was reported independently.

Independent locations do not belong to the track of the user.

void dc_array_t::dc_array_unref(dc_array_t* array)

Free an array object.

Parameters
array The array object to free, created e.g. by dc_get_chatlist(), dc_get_contacts() and so on. If NULL is given, nothing is done.

Does not free any data items.
