char* dc_provider_t::dc_provider_get_before_login_hint(const dc_provider_t* provider)

Get hints to be shown to the user on the login screen.

Parameters
provider The dc_provider_t struct.
Returns A string with the hint to show to the user, may contain multiple lines, if there is no such hint, an empty string is returned, NULL is never returned. The returned value must be released using dc_str_unref().

Depending on the DC_PROVIDER_STATUS returned by dc_provider_get_status(), the UI may want to highlight the hint.

Moreover, the UI should display a "More information" link that forwards to the url returned by dc_provider_get_overview_page().

char* dc_provider_t::dc_provider_get_overview_page(const dc_provider_t* provider)

URL of the overview page.

Parameters
provider The dc_provider_t struct.
Returns String with a fully-qualified URL, if there is no such URL, an empty string is returned, NULL is never returned. The returned value must be released using dc_str_unref().

This URL allows linking to the providers page on providers.delta.chat.

int dc_provider_t::dc_provider_get_status(const dc_provider_t* provider)

Whether DC works with this provider.

Parameters
provider The dc_provider_t struct.
Returns The status as a constant number.

Can be one of DC_PROVIDER_STATUS_OK, DC_PROVIDER_STATUS_PREPARATION or DC_PROVIDER_STATUS_BROKEN.

dc_provider_t* dc_provider_t::dc_provider_new_from_email(const dc_context_t* context, const char* email)

Create a provider struct for the given email address.

Parameters
context The context object.
email The user's email address to extract the provider info form.
Returns a dc_provider_t struct which can be used with the dc_provider_get_* accessor functions. If no provider info is found, NULL will be returned.

The provider is extracted from the email address and it's information is returned.
